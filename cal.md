# UNIX-commands
UNIX and Shell  Programming Lab

### 1.  `Cal`

   ###   `cal` :- Displays a simple , foematted calendar in the Terminal.
   
   ![cal](https://user-images.githubusercontent.com/90957128/157815938-7c9bfdfd-ab76-4ab3-a8ba-15132922686b.png)
   
   ###     Syntax --- 
   
   ###              (i)  `cal -1` :- Display only the current month. This is the default.
   
   ![cal1](https://user-images.githubusercontent.com/90957128/157836859-0bb5c0d6-1bd3-4dfe-9b08-e8e1125dd84e.png)

   ###              (ii)  `cal -3`  :- Display the previous, current and next month surrounding today.
   
   ![cal2](https://user-images.githubusercontent.com/90957128/157836958-d290c0ce-93fe-42f6-8574-101fbfa3532b.png)

   ###              (iii)  `ncal -S` :- Weeks start on Sunday.
   
   ![cal3](https://user-images.githubusercontent.com/90957128/157836999-615f3efc-5ce5-4c1e-b176-930a3010dfa2.png)
   
   ###              (iv)  `ncal -M` :- Weeks start on Monday.
   
   ![cal4](https://user-images.githubusercontent.com/90957128/157837072-3b2bda39-c23a-4be0-b9fc-40a2513a54f5.png)

   ###              (v)  `cal -j` :- Use day-of-year numbering for all calendars. These are also called ordinal days. Ordinal days range from 1 to 366.
  
   ![cal5](https://user-images.githubusercontent.com/90957128/157837298-afc781d9-06b5-4c27-9752-ca16c9c8f995.png)
 
   ###              (vi)  `cal -y` :- Display calendar for whole year.
  
   ![cal6](https://user-images.githubusercontent.com/90957128/157837440-f51e8fa2-8eeb-4aa7-950c-9ca0604b1600.png)
