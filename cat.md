
# UNIX-commands
UNIX and Shell  Programming Lab


### 1.  `cat`

   ###    `cat` :- It is used to create, displat and concatenate file contents.
      
   ![cat](https://user-images.githubusercontent.com/90957128/157866180-08988178-87d9-47d3-a0ae-c21fba1aba6e.png)
  
   ###     Syntax --- cat [option] [file]
     
   ###              (i)  `cat -b` :- Omits line numbers for blank space in the output.
      
   ![cat1](https://user-images.githubusercontent.com/90957128/157865792-6d7ea97e-6054-4339-b646-333b1c45f267.png)

   ###              (ii)  `cat -E`  :- Displays a '$' (dollar sign) at the end of each line.
   
   ![cat2](https://user-images.githubusercontent.com/90957128/157867094-5b0ae087-b1f6-4e35-93d2-34a05150d53b.png)

   ###              (iii)  `cat -n` :- Line numbers for all the output lines.
   
   ![cat3](https://user-images.githubusercontent.com/90957128/157867203-5c3f3b99-244f-4c83-aa34-cc3e8b2bc30e.png)
   
   ###              (iv)  `cat -s` :- Suppress repeated empty output lines.
   
   ![cat4](https://user-images.githubusercontent.com/90957128/157867427-aad3ad75-90da-4a01-ad05-c25b2438dadd.png)
