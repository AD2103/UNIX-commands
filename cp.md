# UNIX-commands
UNIX and Shell  Programming Lab

### 1.  `cp`

   ###   `cp` :- This cammand is used to copy files and directories.
   
   ![cp](https://user-images.githubusercontent.com/90957128/158001904-1176ab65-c281-4ae8-9031-e309b5809fe9.png)
   
   ###     Syntax --- 
   
   ###              (i)  `cp -i` :- Interactive - ask before overwrite .
   
   ![cp1](https://user-images.githubusercontent.com/90957128/158001945-89353e2c-6d90-4f9a-b13d-7025e8281d32.png)

   ###              (ii)  `cp -f`  :- Force - Copy by removing the destination file if needed.
   
   ![cp2](https://user-images.githubusercontent.com/90957128/158001948-07a02bc2-81c6-4d3e-b9fb-75ee796b2581.png)

   ###              (iii)  `cp -n` :- Do not  overwrite  an  existing  file.
      
   ![cp3](https://user-images.githubusercontent.com/90957128/158002006-6ce3c2bb-fa66-4efd-88d2-7a6fda746888.png)
   
   ###              (iv)  `cp -u` :- Update - copy when  the  SOURCE file is newer than the destination file .
   
   
   ###              (v)  `cp -s` :- Make symbolic links instead of copying.
  
   ![cp5](https://user-images.githubusercontent.com/90957128/158002014-a70794ef-5965-4f4c-9dac-6343d448ce54.png)
   
   ###              (vi)  `cp -R` :- Copy directories recursively.
  
   ![cp6](https://user-images.githubusercontent.com/90957128/158002045-a1543dfb-1372-4e01-97bb-222602037777.png)

  ###              (vii)  `cp -v` :- Print informative messages.
  
   ![cp7](https://user-images.githubusercontent.com/90957128/158002047-787ff400-7595-42e2-b5ff-b373d717cb4f.png)
