
# UNIX-commands
UNIX and Shell  Programming Lab


### 1.  `cat`

   ###    `cat` :- It is used to create, displat and concatenate file contents.
      
   ![cat](https://user-images.githubusercontent.com/90957128/157866180-08988178-87d9-47d3-a0ae-c21fba1aba6e.png)
  
   ###     Syntax --- cat [option] [file]
     
   ###              (i)  `cat > file.txt` :-  It creates file.txt and allow us to insert content for this file.
      
  ####    _After inserting content you can use ctrl+c to exit the file._
  
   ![file](https://user-images.githubusercontent.com/90957128/157871091-3ac644d8-9e11-4f7f-95b9-fe7095d4ff73.png)

   ###              (ii)  `cat  file.txt > newfile.txt`  :- Read the contents of file.txt and write them to newfile.txt, overwriting anything newfile.txt previously contained. If newfile.txt does not exist,so it will create that newfile.txt.
   
   ![file1](https://user-images.githubusercontent.com/90957128/157871175-39ceb37d-0a2c-4a11-8fb2-1d5bc02ec2fe.png)

   ###              (iii)  `cat  file.txt ad.txt` :- . It will read the contents of file.txt and ad.txt and display the resulte in the terminal.
   
   ![file2](https://user-images.githubusercontent.com/90957128/157872372-ed10428d-5a98-4480-b29c-270dfee9b36e.png)
   
   ###              (iv)  `cat file.txt ad.txt > combinefile.txt` :- It will concatenate the contents of file.txt and ad.txt and write them to combinefile.txt using the (>) operator .
   ######       If combinefile.txt does not exist the command will create it otherwise it will overwrite the file.
   
   ![file3](https://user-images.githubusercontent.com/90957128/157874732-1942f223-f09f-4519-819a-d270ada3b6e8.png)

   ###              (v)  `cat  file.txt >> newfile1.txt`  :- Read the contents of file.txt and append them to the end of newfile1.txt. If newfile1.txt does not exist, it will be created.
   
   ![file4](https://user-images.githubusercontent.com/90957128/157879711-73d19945-3846-4e10-ae29-6fd6372c8959.png)
