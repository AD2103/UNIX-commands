# UNIX-commands
UNIX and Shell  Programming Lab


### 1.  `mkdir`

   ###    `mkdir` :-  This command is used to make directories.
   
   ![mkdir](https://user-images.githubusercontent.com/90957128/157911130-3c192e6c-7533-4d7d-9507-405251e87678.png)

   ###     Syntax --- 

   ###              (i)  `mkdir -v` :- print a message for each created directory.
   
   ![mkdir1](https://user-images.githubusercontent.com/90957128/157912112-3d698669-903f-430d-898f-08ab40e2a601.png)

   ###              (ii)  `mkdir -p`  :- no error if existing, make parent directories as needed.
   
   ![mkdir2](https://user-images.githubusercontent.com/90957128/157913366-7df3a3c8-15da-42cf-a43a-141782f230fd.png)

   ###              (iii)  `mkdir -m` :- To control the permission of new directories.
    
   ![mkdir3](https://user-images.githubusercontent.com/90957128/157913840-ec9cab34-d097-4172-93f4-72e870ab9634.png)

   
  
### 2.  `rmdir`

   ###   `rmdir` :- This command removes empty directories from your filesystem.
   
   ###     Syntax ---
   
   ###              (i)  `rmdir -p` :- Remove DIRECTORY and its ancestors...  e.g., 'rmdir-p a/b/c' is  similar  to 'rmdir a/b/c a/b a' 
   
  ![rmdir](https://user-images.githubusercontent.com/90957128/157915950-63165965-c435-4341-b34e-a5d7732aadf1.png)
